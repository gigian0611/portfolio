# 🎭 Facial Scanning Project

이 프로젝트는 **메타버스 엔터테인먼트, 넷마블F&C**에서 진행한 **Facial Scanning** 연구입니다.  
얼굴을 스캔하고 보다 **자연스러운 표정 애니메이션**을 구현하는 것이 목표입니다.  
이 기술은 **영상과 게임에 활용**할 수 있도록 개발되었습니다.

## 🛠 사용 기술 (Tech Stack)
- **C++**, **Python**
- **DirectX 12**, **Unreal Engine 5**
